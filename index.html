What Was Fixed?

The issue was in your CSS syntax for the background-image property. Here's the correction:

/* INCORRECT - This was your original code */
background-image: linear-gradient(rgba(0, 0, 0, 0.6)),
    rgba(0,0,0,.6) ,
    url('/hero.jpg');

/* CORRECT - Fixed syntax */
background-image:
    linear-gradient(rgba(0, 0, 0, 0.6), rgba(0, 0, 0, 0.6)),
    url('https://images.unsplash.com/photo-1506744038136-46273834b3fb');

The main issues were:

  • Missing comma between gradient color stops
  • Incorrect placement of the gradient and image declarations
  • Using multiple rgba values incorrectly instead of defining the gradient properly

Now your hero section has a semi-transparent black gradient overlay on top of the background image, creating a professional look with improved text readability.
